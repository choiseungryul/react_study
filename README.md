## React 프로젝트 초기 설정하는 법
1. 프로젝트 생성할 폴더에 가서 shift + 우클릭 => 파워쉘 열기
2. ```npx create-vite 입력```
3. ```Y 입력```
4. 프로젝트 네임 생성
5. ```React 선택```
6. ```JavaScript 선택```
7. 해당 프로젝트 폴더 생성됨
8. vscode에서 오픈폴더해서 해당 프로젝트 폴더를 열기
9. 버전 변경```( dependencies에서 버전을 기본 19.x에서 (package.json -> react : 18.3.1)```
10. 터미널 열기
11. ```npm install``` 입력
12. ```npm run dev``` 입력 ( 확인용 )
13. 취소할 땐 터미널에 ```crtl + c```
